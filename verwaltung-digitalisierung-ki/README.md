# verwaltung-digitalisierung-ki

> Dieses Plugin ersetzt keine rechtliche, datenschutzrechtliche oder
> IT-sicherheitsrechtliche Einzelfallpruefung durch die zustaendige
> Fachstelle und keine verbindliche Entscheidung der zustaendigen Behoerde.
> Die Ausgaben sind Arbeitshilfen ohne Gewaehr. Im Einzelfall ist die
> zustaendige Fachaufsicht, die bzw. der Datenschutzbeauftragte oder die
> Rechtsabteilung zu konsultieren.

Plugin fuer digitale Verwaltung und Staatsmodernisierung: OZG-Umsetzung,
FIT-Connect, FIM-Bausteine, digitale Identitaeten, elektronische
Signaturen/Siegel, KI-Einsatz in Behoerden, Registermodernisierung,
Interoperabilitaet, Barrierefreiheit und Deutschland-Stack.

## Skills

| Skill | Thema |
|---|---|
| `ozg-umsetzung-pruefung` | OZG-Umsetzungsstand einer Verwaltungsleistung |
| `fit-connect-anbindung-checkliste` | Technische/organisatorische Anbindung an FIT-Connect |
| `fim-baustein-modellierung` | Modellierung von FIM-Leistungs-, Prozess- und Datenfeldbausteinen |
| `digitale-identitaet-nutzerkonto-auswahl` | Auswahl von BundID/Unternehmenskonto/eID-Karte |
| `elektronische-signatur-siegel-einordnung` | Einfache, fortgeschrittene oder qualifizierte Signatur/Siegel |
| `ki-einsatz-verwaltungsverfahren-pruefung` | KI-Einsatz in Verwaltungsverfahren nach der EU-KI-Verordnung |
| `registermodernisierung-once-only-pruefung` | Once-Only-Faehigkeit nach dem Registermodernisierungsgesetz |
| `interoperabilitaet-datenaustausch-pruefung` | Technische/semantische Interoperabilitaet, XOEV-Standards |
| `barrierefreiheit-digitale-angebote-pruefung` | Barrierefreiheit nach BITV 2.0/BFSG und WCAG |
| `deutschland-stack-einordnung` | Relevante Basisdienste/Plattformen fuer ein Vorhaben |
| `prozessautomatisierung-verwaltungsleistung-pruefung` | Eignung einer Verwaltungsleistung fuer (Teil-)Automatisierung |
| `eudi-wallet-einsatzpruefung` | Einsatz einer digitalen Wallet (z. B. EUDI-Wallet) fuer Nachweise |
| `datengetriebene-entscheidung-open-data-pruefung` | Voraussetzungen fuer datengetriebene Entscheidungen und Open Data |
| `it-sicherheit-grundschutz-checkliste` | BSI-IT-Grundschutz-Anforderungen bei neuen digitalen Verfahren |
| `datenschutz-folgenabschaetzung-pruefung` | Erforderlichkeit einer Datenschutz-Folgenabschaetzung nach Art. 35 DSGVO |
