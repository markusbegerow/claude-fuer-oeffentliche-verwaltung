# Skills-Index: verwaltung-fachverfahren-kommune

| Skill | Beschreibung |
|---|---|

| `standesamt-eheschliessung-unterlagen-pruefung` | Pruefung der benoetigten Unterlagen fuer eine Eheschliessung oder Geburtsbeurkundung nach dem Personenstandsgesetz |
| `auslaenderbehoerde-aufenthaltstitel-einordnung` | Einordnung des grundsaetzlich einschlaegigen Aufenthaltstitels bzw. Verfahrens nach den Grundzuegen des Aufenthaltsgesetzes |

| `heimat-app-anforderungen-pruefung` | Pruefung fachlicher und technischer Anforderungen an eine kommunale Heimat-App |
| `serviceorientierte-verwaltungsleistung-redesign` | Anleitung zum nutzerzentrierten Redesign einer bestehenden Verwaltungsleistung |
| `digitales-bildungswesen-schulverwaltung-pruefung` | Pruefung des Digitalisierungsstands von Schulverwaltungsprozessen (z. B. Schulanmeldung, Lernplattform-Anbindung, Schulverwaltungssoftware) eines Schultraegers bzw. einer Kommune |
