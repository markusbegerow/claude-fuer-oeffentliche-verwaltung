# Quickstart (2 Minuten)

1. Repository in Claude Code oeffnen (Ordner als Projektverzeichnis
   waehlen).
2. Claude Code erkennt die drei Plugins automatisch ueber
   `.claude-plugin/marketplace.json`:
   - `verwaltung-digitalisierung-ki`
   - `verwaltung-fachverfahren-kommune`
   - `verwaltung-organisation-sicherheit`
3. Skill aufrufen, z. B.: "Nutze den Skill
   `ozg-umsetzung-pruefung`, um folgenden Sachverhalt zu pruefen: ..."
4. Sachverhalt (fiktive Beispieldaten oder eigene, anonymisierte Angaben)
   mitliefern.
5. Ergebnis erhalten — **immer mit Disclaimer und Stand-Datum**.

## Wichtig vor dem ersten Einsatz

- `CLAUDE.md` und `AGENTS.md` einmal lesen — sie definieren, wie Outputs
  aufgebaut sind und welche Disclaimer enthalten sein muessen.
- Keine echten Buerger-, Mitarbeiter- oder Aktendaten eingeben.
- Ausgaben sind Arbeitshilfen, keine Rechtsberatung und keine verbindliche
  Behoerdenentscheidung.

Ausfuehrlichere Schritt-fuer-Schritt-Anleitung: siehe
`INSTALLATION_EINFACH.md`.
